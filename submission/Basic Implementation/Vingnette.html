Bootstrap Estimation for Boston Housing

This example demonstrates how to perform bootstrap estimation for Simple Linear Regression (SLR) using the Boston Housing Dataset.

We use the response variable medv and the predictor variable lstat as an example. We will generate a linear regression model first and then use bootstrap resampling to estimate the sampling variability, bias, and confidence intervals of the intercept and slope.

data <- read.csv("BostonHousing.csv")
source("main.R")

# Example variables
respond <- "medv"
predictor <- "lstat"

The analysis is implemented through the following functions:
- ols_estimators(data, predictor, respond)
- bootstrap_slr_summary(data, R, seed, predictor, respond)
- bootstrap_slr(boot_slr, ols_slr)
- bootstrap_slr_ci(boot_slr, ols_slr, level)
- plot_boot_hist(boot_slr, ols_slr, term = c("intercept","slope"))
- plot_ci_box(ci_list, term = c("intercept","slope"))
- plot_lr_bootstrap_scatter(ci_list, term = c("intercept","slope"))

Bootstrap for Simple Linear Regression

Bootstrap estimation

We run bootstrap with R = 1000 resamples.

boot_slr <- bootstrap_slr_summary(
  data = data,
  R = 1000,
  seed = 123,
  predictor = predictor,
  respond = respond
)

Comparison of Bootstrap estimation and OLS estimation

This figure visualizes the joint sampling distribution of the intercept and slope using the Bootstrap method with 1000 resamples. We can identify the following message:

Parameter correlation: The tilt direction of the point cloud shows the trade-off between the intercept and slope.

Bias magnitude: The arrow length reflects the degree of deviation between the Bootstrap mean and the OLS estimate.

Model stability: The dispersion of the point cloud corresponds to the variance of the parameter estimates.

Estimation reliability: Whether the orange rhombus falls within a high-density region indicates the representativeness of the original estimate.
