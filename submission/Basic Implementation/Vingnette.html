Bootstrap Estimation for Boston Housing

Overview

This vignette demonstrates bootstrap estimation for a simple linear regression (SLR) using the Boston Housing dataset. We model the relationship between the response variable medv and the predictor lstat, and then use bootstrap resampling to estimate sampling variability, bias, and confidence intervals for the intercept and slope.

Workflow: 1) fit OLS SLR and obtain standard (Wald) confidence intervals
2) bootstrap the SLR coefficients and summarize bias/SE/variance/MSE
3) compare bootstrap percentile CIs to OLS CIs
4) check standard linear regression assumptions with diagnostic plots

Data

The Boston Housing dataset contains housing and neighborhood variables for census tracts in the Boston area (see (Harrison Jr and Rubinfeld 1978)).

Functions and Usage

All group functions are sourced from main.R. Key functions used here include:

  • ols_estimators(data, predictor, respond)
  • bootstrap_slr_summary(data, R, seed, predictor, respond)
  • bootstrap_slr(boot_slr, ols_slr)
  • bootstrap_slr_ci(boot_slr, ols_slr, level)
  • plot_boot_hist(boot_slr, ols_slr, term = c("intercept","slope"))
  • plot_ci_box(ci_list, term = c("intercept","slope"))

Bootstrap + Linear Regression Assumptions

Bootstrap estimation

We run bootstrap with R = 1000 resamples (see (A. C. Davison and Hinkley 1997; Canty, Ripley, and Brazzale 2025)).

boot_slr <- bootstrap_slr_summary(
  data = data,
  R = 1000,
  seed = 123,
  predictor = predictor,
  respond = respond
)
names(boot_slr)
#> [1] "boot_b0_star" "boot_b1_star" "boot_b0_sd"   "boot_b1_sd"   "boot_b0_bias"
#> [6] "boot_b1_bias" "boot_b0_ci"   "boot_b1_ci"
